# Nongdam Frontend Project

React Native와 TypeScript를 사용하여 개발하는 Nongdam 팀의 프론트엔드 레포지토리입니다.

## 🚀 프로젝트 소개
*   **플랫폼**: React Native (Android / iOS)
*   **언어**: TypeScript
*   **주요 라이브러리**: React Navigation, Zustand (추천)

---

## 🛠️ 개발 규칙 및 가이드라인
일관성 있고 유지보수하기 쉬운 코드베이스를 위해 사전에 정의된 규칙과 컨벤션을 반드시 준수해 주세요.

*   [폴더 구조 및 코드 네이밍 컨벤션 가이드](file:///c:/Users/SSAFY/GitHub/frontend/rules/convention.md)

## ⚙️ 실행 및 빌드 방법 (준비 중)

```bash
# 의존성 패키지 설치
npm install

# iOS 실행
npm run ios

# Android 실행
npm run android
```
